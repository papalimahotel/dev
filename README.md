web-stack
===
**web-stack** is a containerized bundle for a LAMP stack for web projects.  The goal is to have a bundle that portable and platform agnostic.  

## Technology
* Ubuntu v18.04
* Apache v2.4
  * libapache2-mod-php7
  * libapache2-mod-xsendfile
* PHP v7.4.4
  * php-ldap
  * php-gd
  * php-xml
  * php-curl
  * php-readline
  * php-json
  * php-mbstring
  * php-mysql
* MariaDB Server v10.0
